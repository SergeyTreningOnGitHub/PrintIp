Обобщенная функция печати ip адреса в поток вывода.
ip адрес может принадлежать к следующим типам:
любой интегральный тип (char, short, int и т.д.),
vector<T>, где Т любой интегральный тип,
list<T>, где T любой интегральный тип,
string,
tuple<Types...>, где Types... любые интегральные типы и Types... - одинаковые типы.
Если тип ip адреса не относится к вышеперечисленным, то будет вызвана ошибка компиляции.
